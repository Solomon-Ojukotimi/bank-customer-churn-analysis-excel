Bank Customer Churn Analysis (Excel Dashboard)
# Project Overview

This project analyzes customer churn in a retail banking environment using Microsoft Excel.
The goal is to identify high-risk customer segments, understand churn drivers, and provide data-driven insights to improve customer retention strategies.

The analysis is presented through interactive Excel dashboards designed for executive and stakeholder decision-making.

# Tools & Skills Used

Advanced Excel formulas

Business insight storytelling

# Key Business Questions Answered

Which customer segments are most likely to churn?

How does churn vary by age, gender, location, and tenure?

Does credit card ownership reduce churn?

Which balance ranges contribute most to revenue loss?

How does customer activity level impact churn?
